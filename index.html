Welcome

Welcome to my portfolio. Please look around, play with the games, and try and break the apps. If you have any questions or comments, please don't hesitate to contact me. You can see all of the source code and the rest of my projects on GitHub and CodePen. If you want to see what I am currently building, please following me on Twitter.

Moog Drum Machine

Built with React, the Moog Drum Machine allows you to combine and alter the sounds from your favorite synthesizer. Choose between two sound banks, as well as adjustments for volume and tempo.

Built With:

HTML5, CSS3, JavaScript, and React

Top Ten Jukebox

Allows you to see and play the Billboard top ten songs from any date. Integrated with Spotify to allow you to save any of the songs to a new or existing playlist.

Built With:

Node.js, Express.js, HTML5, CSS3, JavaScript, jQuery, and Spotify API

Dynamic Newspaper

Allows you to see all of the lastest news in one place. The dynamic newspaper will load the lastest headlines upon refresh. Click on the article to view story from original source.

Built With:

HTML5, CSS3, JavaScript, and jQuery

Analog Radio

Listen to radio the way it was meant to be, with the Analog Radio. Hold the mouse down over the knob to manually tune, or click one of the five buttons to go to a preset station.

Built With:

HTML5, CSS3, JavaScript, and jQuery

Pillow Real Estate

Built with React, the Pillow real estate portal allows you to search and view homes for sale or rent. The design is based on Zillow.com

Built With:

Node.js, Express.js, HTML5, CSS3, JavaScript, React, and Google Maps API

Hans' Boathouse

Website created for Brussels seafood restaurant, Hans' Boathouse. Responsive layout created with CSS Grid.

Built With:

HTML5, CSS3, Bootstrap, JavaScript, and jQuery

Retro YouTube Television

Want to channel surf with YouTube videos? Now you can with the Retro YouTube Television! Each channel plays a different category of video. Check the channel guide for complete lineup.

Built With:

HTML5, CSS3, JavaScript, jQuery and YouTube API
