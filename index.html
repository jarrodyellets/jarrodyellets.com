Me

My name is Jarrod Yellets and I am a web developer residing in Brussels, Belgium. I specialize in front-end development with working knowledge of back-end technologies such as node.js, express.js, and mongoDB. I have always viewed coding as an art form, thus we should be continuously pushing the boundary while having the end product functional and intuitive. When I'm not behind the keyboard, you can find me running, boxing, chasing my Golden Retriever, Padme, or traveling Europe with my wife and daughter.

Simon Game

Blast from the past! Play your favorite childhood game with Simon. Features running count current sequence, custom sounds, and strict mode, which offers a no mercy version of the game.

Built With:

HTML5, CSS3, JavaScript, and jQuery

Top Ten Jukebox

Allows you to see and play the Billboard top ten songs from any date. Integrated with Spotify to allow you to save any of the songs to a new or existing playlist.

Built With:

Node.js, Express.js, HTML5, CSS3, JavaScript, jQuery, and Spotify API

Dynamic Newspaper

Allows you to see all of the lastest news in one place. The dynamic newspaper will load the lastest headlines upon refresh. Click on the article to view story from original source.

Built With:

HTML5, CSS3, JavaScript, and jQuery

Analog Radio

Listen to radio the way it was meant to be, with the Analog Radio. Hold the mouse down over the knob to manually tune, or click one of the five buttons to go to a preset station.

Built With:

HTML5, CSS3, JavaScript, and jQuery

Lightsabers

Pick one. But beware, the dark side of the force clouds everything. Click on hilt to activate lightsaber.

Retro YouTube Television

Want to channel surf with YouTube videos? Now you can with the Retro YouTube Television! Each channel plays a different category of video. Check the channel guide for complete lineup.

Built With:

HTML5, CSS3, JavaScript, jQuery and YouTube API
